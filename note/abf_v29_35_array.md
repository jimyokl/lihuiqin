4、指针数组 vs 数组指针
    int *p[3]，[]的优先级高于*，会先执行数组声明，然后 再解释 每个元素的类型是int *(指向整数的指针)
    这是一个指针数组，包含 3 个元素
    eg,
    int a = 1, b = 2, c = 3;
    int *p[3] = { &a, &b, &c };  // 三个 int 指针组成的数组


    int (*p)[3]是一个指向包含3个int的数组的指针，
    eg,
    int arr[2][3] = { {1, 2, 3}, {4, 5, 6} };
    int (*p)[3] = arr;   // 指向第一个子数组，即 arr[0]
    (*p)[1] 是第 2 个元素（即 arr[0][1]）
